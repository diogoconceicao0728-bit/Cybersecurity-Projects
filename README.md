Cybersecurity Projects

Este repositório reúne os projetos que venho desenvolvendo ao longo da minha formação em Segurança Cibernética. A ideia aqui é manter um registro claro e organizado da minha evolução prática, documentando o que estudo, o que aplico e as soluções que construo para treinar habilidades técnicas.

O conteúdo é atualizado aos poucos, conforme avanço nos estudos e nos projetos pessoais.

1. Monitor de Tentativas de Acesso (Python + PowerShell)

Ferramenta criada para analisar eventos de logon do Windows (4624, 4625 e 4634).
O projeto converte os registros para JSON estruturado, mantém arquivos rotacionados e facilita a leitura e investigação dos padrões de autenticação.

É um ponto de partida sólido para estudos voltados a SIEM, análise de logs e resposta a incidentes.
Os arquivos serão adicionados conforme avanço no refinamento da ferramenta.

2. Hardware Hacking — Alarme de Intrusão (Arduino/IoT)

Projeto desenvolvido para explorar princípios de segurança em dispositivos físicos.
O sistema simula um alarme simples utilizando push button, LEDs e buzzer, permitindo observar a lógica de acionamento e resposta em um ambiente IoT básico.

Incluirei o código e os detalhes da montagem nas próximas atualizações.

3. Scripts de Segurança em Python

Conjunto de scripts que desenvolvo para treinar automação e lógica aplicada à segurança.
Incluo aqui pequenos projetos como:

validação e sanitização de entradas

leitura e processamento de logs

testes educativos relacionados a injeção

automações administrativas

Cada script reflete um conceito que estou estudando ou validando na prática.

4. Web Security Labs / CTFs

Anotações e resoluções de desafios realizados em ambientes controlados.
Uso essa seção para praticar e revisar vulnerabilidades clássicas, entender respostas de servidor e aprimorar o raciocínio de exploração.

Alguns dos tópicos abordados:

SQL Injection

XSS

enumeração e mapeamento

coleta e análise de informações

Todo o material tem finalidade exclusivamente educacional.

5. Estudos OWASP

Documentação dos estudos relacionados ao OWASP Top 10.
Reúno exemplos, explicações e pequenos testes voltados a entender a lógica de cada vulnerabilidade e suas formas de mitigação.

Este conteúdo cresce à medida que avanço nos estudos.
